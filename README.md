# Gazua

암호화폐 시세 확인 웹서비스 'Gazua'



### 구성

- [ ] 코인 리스트 페이지: 코인들의 간단한 정보와 현재 가격
- [ ] 코인 상세 페이지: 코인의 상세 정보와 가격에 대한 상세 정보





### 기술

React, Redux
